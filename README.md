## 📱 Installation

1. **Download** the latest APK from releases
2. **Enable** "Install from Unknown Sources" on your Android device
   - Settings → Security → Unknown Sources (allow)
3. **Install** the downloaded APK
4. **Launch** Real Racing 3 Community Edition
5. **Select** a community server from the server browser
6. **Race!** 🏁

## 📋 Version Scheme

**Format:** `MAJOR.MINOR.PATCH-channel`

**Example:** `15.0.0-community-alpha`
- **15.0.0** - Version number (continuing from EA's 14.x)
- **community** - Community edition identifier
- **alpha/beta/stable** - Release channel

**Version Codes:**
- Format: Integer representation (e.g., 150000 for 15.0.0)
- Used for update comparisons
